-> What is Tags :- "like Bookmark in Book"

- **A Branch**  is like a bookmark that moves. Every time you write a new page, you move the bookmark to the very end of the book to keep tracking your current progress.

- **A Tag**  is a sticky note glued to a page. It never moves. It stays on that specific page forever to mark a special moment.

###  Why do developers use them?

- To mark releases: To easily find "Version 1.0", "Version 2.0", etc.

- For safety: If a new update breaks everything, you can instantly go back to the safely tagged version.

- For teamwork: Everyone on the team knows exactly which version of the code is currently live for users.
